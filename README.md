# Título del Proyecto

Nombre del Proyecto: CourseRobotFrontend

## Descripción

CourseRobotFrontend es una aplicación web diseñada para gestionar cursos en línea. Permite a los usuarios crear, editar y eliminar cursos, así como gestionar usuarios y actividades relacionadas.

## Características

- Gestión de cursos: Crear, editar y eliminar cursos.
- Gestión de usuarios: Administrar perfiles de usuario y permisos.
- Actividades interactivas: Integración de videos, documentos y cuestionarios.
- Interfaz intuitiva: Diseño amigable y fácil de usar.

## Instalación

Para instalar y ejecutar el proyecto, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone <URL del repositorio>
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd CourseRobotFrontend
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

4. Inicia la aplicación:
   ```bash
   npm run dev
   ```

## Uso

Una vez que la aplicación esté en funcionamiento, puedes acceder a ella en tu navegador en `http://localhost:3000`. Desde allí, podrás gestionar cursos y usuarios.
