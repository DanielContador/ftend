# Título del Proyecto

Nombre del Proyecto: CourseRobotFrontend

## Descripción y Alcance

CourseRobotFrontend es una aplicación web diseñada para gestionar cursos en línea. El objetivo del proyecto es desarrollar un sistema que permita a los usuarios crear, editar y gestionar cursos de manera eficiente. Este sistema utiliza inteligencia artificial para generar la estructura del curso y sus contenidos, facilitando así la creación de cursos interactivos y personalizados. El sistema debe ser intuitivo y fácil de usar, permitiendo a los educadores y administradores gestionar el contenido del curso y las actividades asociadas.

### Levantamiento de Requisitos
#### Requisitos Funcionales
- Los usuarios deben poder crear nuevos cursos.
- Los usuarios deben poder editar la información de los cursos existentes.
- El sistema debe permitir la gestión de la estructura del curso.
- Los usuarios deben poder generar actividades asociadas a los cursos.

#### Requisitos No Funcionales
- La aplicación debe ser responsiva y accesible desde dispositivos móviles.
- El sistema debe garantizar la seguridad de los datos de los usuarios.

## Características

- Gestión de cursos: Crear, editar y eliminar cursos.
- Gestión de usuarios: Administrar perfiles de usuario y permisos.
- Actividades interactivas: Integración de videos, documentos y cuestionarios.
- Generación de la estructura de cursos y contenidos utilizando inteligencia artificial.
- Interfaz intuitiva: Diseño amigable y fácil de usar.


## Definición de Estructura del Curso
La estructura del curso se define mediante módulos que contienen diferentes actividades de aprendizaje. Cada módulo puede ser expandido o colapsado para mostrar u ocultar las actividades asociadas.

## Generación de Actividades
Las actividades se generan automáticamente en función de la estructura del curso definida por el usuario. Los educadores pueden personalizar las actividades según las necesidades del curso.

## Instalación

Para instalar y ejecutar el proyecto, sigue estos pasos:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/DigitalLearning/CourseRobotFrontend
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd CourseRobotFrontend
   ```

3. Instala las dependencias:
   ```bash
   npm install
   ```

4. Inicia la aplicación:
   ```bash
   npm run dev
   ```

## Uso

Una vez que la aplicación esté en funcionamiento, puedes acceder a ella en tu navegador en `http://localhost:3000`. Desde allí, podrás gestionar cursos y usuarios.
